# 模拟停车场出入场系统

这是一个按课程实验要求实现的最小可运行版本：
- 手机网页拍照/选图上传车牌图片
- Flask 服务端接收图片
- EasyOCR 识别车牌号
- SQLite 保存入场/出场记录
- 自动计算停车时长和费用

## 1. 安装依赖

建议 Python 3.10+。

```bash
pip install -r requirements.txt
```

## 2. 启动项目

```bash
python app.py
```

启动后访问：
- 本机浏览器：`http://127.0.0.1:5000`
- 手机浏览器：`http://你的电脑局域网IP:5000`

例如电脑 IP 是 `192.168.1.23`，则手机访问：

```text
http://192.168.1.23:5000
```

## 3. 手机访问不到时检查

1. 手机和电脑是否连接同一个 Wi‑Fi
2. Flask 是否绑定了 `0.0.0.0`
3. 系统防火墙是否放行 5000 端口

## 4. 收费规则

- 停车 30 分钟内免费
- 超过 30 分钟后，每小时 5 元
- 不满 1 小时按 1 小时计费

## 5. 数据库位置

SQLite 数据库保存在：

```text
database/parking.db
```

## 6. 说明

- 如果 EasyOCR 识别效果不理想，可以在 `ocr_utils.py` 里替换成百度 OCR 或其他在线服务。
- 现在默认使用本地 EasyOCR，`gpu=False`，普通电脑可运行。
- 上传图片会保存在 `uploads/` 目录。

## 7. 课程报告可写的关键点

- H5 客户端 + Flask 服务端的 C/S 架构
- HTTP + multipart/form-data 上传图片
- JSON 响应结果
- OCR 模块识别车牌
- SQLite 持久化停车记录
- 异常处理：OCR 失败、重复入场、未入场先出场
